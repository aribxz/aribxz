# 🏠 Ames Housing Price Prediction System

**Production-ready Machine Learning Regression Pipeline**

</summary>

### Overview

A complete end-to-end machine learning project focused on predicting residential property prices using the **Ames Housing Dataset**. The project explores multiple regression algorithms, extensive feature engineering, hyperparameter tuning, and rigorous model evaluation to identify the most effective predictive model.

Rather than relying on a single algorithm, the project compares several regression techniques to understand their strengths, limitations, and generalization performance on real-world housing data.

| **Dataset** | Ames Housing (2,930 houses, 80+ features) |

| **Performance** | **R² = 0.88** |
| **Focus** | Regression, Feature Engineering, Model Selection |
| **Status** | ✅ Completed |

---

- Residual analysis
- Model comparison framework

**Machine Learning + Football Analytics Platform**

</summary>

### Overview

An ongoing football analytics project designed to predict **Home Win**, **Draw**, or **Away Win** outcomes before kickoff using advanced machine learning techniques and statistical modeling.

The project has gradually evolved from classical decision trees into a hybrid prediction system combining ensemble learning with probabilistic football models.

Special emphasis has been placed on preventing data leakage through chronological validation and realistic feature engineering.

| **Status** | 🚧 Continuously Improving |

# 🤖 HAS (Hybrid Assistant System)

**Enterprise AI Knowledge Platform**

</summary>

HAS is my most ambitious ongoing project.

It is a modular AI assistant designed to transform long-form educational content into structured, searchable, high-quality study material.

Instead of acting as a chatbot, HAS focuses on building an intelligent knowledge pipeline capable of processing YouTube lectures, PDFs, presentations, and documents into well-organized notes.

The project emphasizes scalable software architecture, modular design, and production-quality engineering principles.

| **Status** | 🚧 Currently Building |
